# Contributing to ADK MongoDB Session

First off, thank you for considering contributing! Your help is appreciated.

## How Can I Contribute?

### Reporting Bugs

If you find a bug, please open an issue. Be sure to include:

*   A clear and descriptive title.
*   A detailed description of the problem, including steps to reproduce it.
*   Any relevant code snippets, error messages, or logs.
*   The version of the package you are using.

### Suggesting Enhancements

If you have an idea for an enhancement, please open an issue to discuss it. This allows us to coordinate efforts and ensure the change aligns with the project's goals.

### Pull Requests

Pull requests are welcome! For major changes, please open an issue first to discuss what you would like to change.

When submitting a pull request, please make sure:

1.  Your code follows the existing style and conventions.
2.  You have added or updated tests to cover your changes.
3.  All tests pass.
4.  Your commit messages are clear and descriptive.

We appreciate your contributions!
